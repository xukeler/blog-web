    # 手机号授权登录

    • 微信小程序一般采用授权手机号登录，这样比较便捷，对用户体验比较友好，实现也比较简单。
    • 在获取用户手机号之前必须先调uni.login(),获取一个code，提供给后端。
    • 手机号授权只能通过button组件的open-type=getPhoneNumber触发@getphonenumber回调函数，返回加密数据encryptedData,需要传递给后端进行登录注册操作。open-type的类型有很多，参考官方文档。
    • 初始进入页面获取code存到本地，一起传给后端
    • 回调返回值存在encryptedData代表用户同意授权获取手机号，否则是拒绝，要处理好拒绝的逻辑。

    提示

    进入页面必须先调uni.login(),如果getphonenumber回调中调用uni.login登录，可能会刷新登录态。此时服务器使用 code 换取的 sessionKey 不是加密时使用的 sessionKey，导致解密失败。

    # button组件

          <button
        class="phone-login-btn"
        type="default"
        open-type="getPhoneNumber"
        @getphonenumber="getphonenumber"
      >
        <text class="btn-text">手机号登录</text>
      </button>

    # 回调函数getphonenumber

        getphonenumber(res) {
      if (res.detail.encryptedData) { //同意获取手机号
        uni.showLoading({
          title: "登录中...",
          mask: true,
        });
        uni.checkSession({
          success: () => {
            // 调登录接口
          },
          fail: () => {
            uni.login({
              success: (result) => {
                this.loginCode = result.code;
                // 登录
              },
              fail: () => {
                util.toastMsg("登录失败");
              },
            });
          },
        });
      } else { //拒绝获取手机号
        let url = "/pages/projects/pages/quickLogin";
        if (this.libid || this.copyid) {
          url += `?libId=${this.libid}&copyId=${this.copyid}`;
        }
        uni.navigateTo({ url });
      }
    },

    # 回调函数

        wxLogin(res) {
      var userInfo = res.detail;
      if (userInfo.errMsg !== "getUserInfo:ok") return;
      uni.showLoading({
        title: "登录中...",
        mask: true,
      });
      uni.login({
        success: (res1) => {
          if (!res1.code) {
            util.toastMsg("获取用户登录状态失败！");
            return;
          }
          uni.getUserInfo({
            success: (res) => {
              userInfo = res;
              let params = {
                v_code: res1.code,
                rawData: userInfo.rawData,
                encryptedData: userInfo.encryptedData,
                iv: userInfo.iv,
              };
            //调接口
            },
          });
        },
      });
    },
