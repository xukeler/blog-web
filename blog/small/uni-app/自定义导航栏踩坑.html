    # 自定义导航栏

    • 小程序的顶部导航栏是可以自己定义的，最近开发的时候用到这个东西，也发现了一些兼容性问题。
    • 第一个问题就是每个端对自定义导航栏的字体样式渲染不一样，像QQ小程序渲染的字体就会加粗加大，像头条小程序，飞书小程序渲染的字体就会偏小不加粗。
    • 微信小程序的安卓端标题靠左，别的安卓端标题都是剧中，就微信例外，所以还要为它单独处理。
    • 百度小程序的自定义导航原生的返回按钮竟然没有移除，也要单独处理否则会显示两个自定义导航，把自定义的返回按钮给隐藏。
    • 微信小程序的pc版本不支持自定义导航你敢信，设置的自定义导航会下来显示到内容区域，不支持也就算了，它还把原生的隐藏了，导致无法适配，如果非要适配就只能写一套正常的页面，在路由跳转的时候判断是不是微信pc版，再跳转到对应页面，缺点就是用到自定义导航的页面很多，要写很多个页面，维护很麻烦，不建议这样做。
    • 这些问题只有在真机测试才能发现，模拟器看不出来，测试的时候也要注意。

    # 开发

    • 自定义导航栏用的是uni-app的官方插件uni-nav-bar这个插件，这个插件很一般因为懒得自己写，就拿过来用了，使用才发现真的很一般，没有对不同平台做字体的适配，以及百度平台的原生按钮问题，安卓微信标题问题做适配。 所以要自己动手改，修改也没什么技术含量，针对不同的平台做适配。

<script>
	import statusBar from "./uni-status-bar.vue";
	const app=getApp()
	/**
	 * NavBar 自定义导航栏
	 * @description 导航栏组件，主要用于头部导航
	 * @tutorial https://ext.dcloud.net.cn/plugin?id=52
	 * @property {String} title 标题文字
	 * @property {String} leftText 左侧按钮文本
	 * @property {String} rightText 右侧按钮文本
	 * @property {String} leftIcon 左侧按钮图标（图标类型参考 [Icon 图标](http://ext.dcloud.net.cn/plugin?id=28) type 属性）
	 * @property {String} rightIcon 右侧按钮图标（图标类型参考 [Icon 图标](http://ext.dcloud.net.cn/plugin?id=28) type 属性）
	 * @property {String} color 图标和文字颜色
	 * @property {String} backgroundColor 导航栏背景颜色
	 * @property {Boolean} fixed = [true|false] 是否固定顶部
	 * @property {Boolean} statusBar = [true|false] 是否包含状态栏
	 * @property {Boolean} shadow = [true|false] 导航栏下是否有阴影
	 * @event {Function} clickLeft 左侧按钮点击时触发
	 * @event {Function} clickRight 右侧按钮点击时触发
	 * @event {Function} clickTitle 中间标题点击时触发
	 */
	export default {
		name: "UniNavBar",
		data(){
			return {
				isBaidu:false,
				isAndroid:false,
				isWindow:false,
			}
		},
		components: {
			statusBar
		},
		props: {
			title: {
				type: String,
				default: ""
			},
			leftText: {
				type: String,
				default: ""
			},
			rightText: {
				type: String,
				default: ""
			},
			leftIcon: {
				type: String,
				default: ""
			},
			rightIcon: {
				type: String,
				default: ""
			},
			fixed: {
				type: [Boolean, String],
				default: true
			},
			color: {
				type: String,
				default: "#111111"
			},
			backgroundColor: {
				type: String,
				default: "white"
			},
			statusBar: {
				type: [Boolean, String],
				default: false
			},
			shadow: {
				type: [Boolean, String],
				default: false
			},
			border: {
				type: [Boolean, String],
				default: true
			},
			androidTitleSite:{//默认安卓标题靠左
				type:Boolean,
				default:true
			}
		},
		created(){
			this.setData({
				isBaidu:app.globalData.isBaidu //百度小程序隐藏自定义
			})
			this.checkSystem()
		},
        mounted() {
          if(uni.report && this.title !== '') {
              uni.report('title', this.title)
          }
        },
		computed:{
			androidTitleState(){
				if(!this.isAndroid) return false;
				return this.androidTitleSite
			}
		},
		methods: {
			checkSystem(){
				let systemInfo=uni.getSystemInfoSync();
				if(systemInfo){
					const {system}=systemInfo;
					this.setData({
						isAndroid:system.indexOf('Android')>-1?true:false,
						isWindow:system.indexOf('Windows')>-1?true:false,//window 提高导航栏
					})
					
				}
			},
			onClickLeft() {
				this.$emit("clickLeft");
			},
			onClickRight() {
				this.$emit("clickRight");
			},
			onClickTitle() {
				this.$emit("clickTitle");
			}
		}
	};
</script>

<script>
	import icons from './icons.js';
	// #ifdef APP-NVUE
	var domModule = weex.requireModule('dom');
	
	// #endif

	/**
	 * Icons 图标
	 * @description 用于展示 icons 图标
	 * @tutorial https://ext.dcloud.net.cn/plugin?id=28
	 * @property {Number} size 图标大小
	 * @property {String} type 图标图案，参考示例
	 * @property {String} color 图标颜色
