    # node.js

    # 非阻塞IO

    • IO是指计算机的输入和输出，例如磁盘拷贝文件，储存文件。
    • 阻塞IO是指主进程在执行时，发出一个IO指令交给系统底层去处理，系统处理过程中，主进程会一直等待，直到处理结束，继续向下执行。
    • 非阻塞IO就是指主进程在执行时，发出一个IO指令交给系统底层去处理，系统处理过程中，主进程不会等待继续向下执行，IO处理结束会通知主进程。

    # 事件驱动

    • IO执行结束之后通知主进程，就是利用事件驱动。
    • 事件驱动就是像前端实现的点击事件一样，每个流程都有对应的事件，执行到该流程，就会触发事件，所以说是事件驱动。

    # 模块化

    • 在node中主要构成部分就是模块，例如http模块，path模块等。使用这些模块必须遵循CommonJS规范。
    • CommonJS规范
      • 所有代码都运行在模块作用域，不会污染全局作用域。
      • 模块可以多次加载，但是只会在第一次加载时运行一次，然后运行结果就被缓存了，以后再加载，就直接读取缓存结果。要想让模块再次运行，必须清除缓存。
      • 模块加载的顺序，按照其在代码中出现的顺序。
    • 可以把公共的功能 抽离成为一个单独的js文件 作为一个模块，默认情况下面这个模块里面的方法或者属性，外面是没法访问的。如果要让外部可以访问模块里面的方法或 者属性，就必须在模块里面通过exports 或者 module.exports 暴露属性或者方法。
    • 在需要使用这些模块的文件中，通过 require 的方式引入这个模块。这个时候就可以 使用模块里面暴露的属性和方法。
    // http.js
function checkDate(){
    return new Date()
}
module.exports=checkDate


// index.js
var date=require("./http")
console.log(date())

    # node的三类模块

    • 全局模块，不需要引入可直接使用，例如process模块。
    • 核心模块，不需要在npm下载，通过require方法引入，引入不带有路径会在nodemodules目录下查找，常见的核心模块http模块，path模块，fs等模块。
    • 自定义模块，是自己定义的文件，可以是js文件，也可以是json文件。

    # path模块

    • 路径相关

      • normalize() 规范化给定的path。
      • join()将path连接在一起。
      • resolve()解析为绝对路径
      • isAbsolute()检查当前路径是否为绝对路径。

    • 文件相关

      • basename返回路径中最后一部分的文件名。
      • extname返回路径最后文件名的扩展名。
      • dirname返回路径中的目录名。

    • 路径解析

      • parse返回一个对象，属性表示路径的有效元素。
      • format把对象转化为路径字符串。
    
var {normalize,join,resolve,isAbsolute,basename,extname ,dirname,parse,format}=require("path")
console.log(normalize("./a/b\\d/f/c.text"))  //   a\b\d\f\c.text
console.log(join("./a/b","/c.text"))  //   a\b\c.text
console.log(resolve("./a/b/c.text")) //     D:\Demo\koa2\a\b\c.text
console.log(isAbsolute("./a/b/c.text"))//  false
console.log(isAbsolute("/a/b/c.text"))//  true


console.log(basename("./a/b/c.text"))  //  c.text
console.log(extname("./a/b/c.text"))//    .text
console.log(dirname("./a/b/c.text"))  // ./a/b


console.log(parse("./a/b/c.text"))  //  { root: '', dir: './a/b', base: 'c.text', ext: '.text', name: 'c' }
let obj={
    root: '', dir: './a/b', base: 'c.text', ext: '.text', name: 'c'
}
console.log(format(obj))  //./a/b/c.text
