    # 动画animation

    • css3新增了动画animation，主要是能在开发中添加动态效果，让界面动起来，在交互方面提升趣味性，之前做一个动画效果，需要通过js来实现，实现起来也比较麻烦，如果要复用动画效果，需要给不同的元素绑定动画事件，css3的animation通过css来实现动画效果，实现起来简单很多，动画效果也比较丰富，复用起来也比较方便。

    # 语法

    • 使用动画首先要定义一个动画，通过@keyframes animationName{}来定义动画，{}内部是动画结构,能够实现的动画效果例如颜色变化color background-color,字体大小变化font-size,边框变化border，尺寸变化width height,间距变化margin padding top left right bottom等效果。

    • 动画结构可以使用from{}或百分比10%{}来定义动画的关键帧。

    • 定义好的动画使用animation属性在css中使用。

    • animation有八个属性配置。

      • name动画名，@keyframes的动画名称

      • duration动画时长，以秒或毫秒为单位。

      • timing-function动画运行效果。

        • linear动画从头到尾的速度是相同的。
        • ease先加速再加速。
        • ease-in逐渐加速。
        • ease-out逐渐减速。
        • ease-in-out低速开始和结束。

      • delay动画延迟执行时间,以秒或毫秒为单位。

      • iteration-count动画播放次数。

        • n一个数字表示播放的次数。
        • infinite表示一直播放。

      • direction是否反向播放动画。

        • normal正常播放。
        • reverse反向播放。
        • alternate奇数正向播放，偶数反向播放。
        • alternate-reverse奇数反向播放，偶数正向播放。

      • fill-mode动画播放结束时的状态，是显示播放之前的关键帧，还是播放结束的关键帧。

        • none什么状态都没有。
        • forwards播放结束显示播放结束的关键帧。
        • backwards播放结束显示播放之前的关键帧。
        • both和 forwards表现出来的效果相同。

      • play-state动画的状态是停止还是播放。

        • running允许播放动画。
        • paused停止动画，不能播放。

    • 不必要的属性可以省略。

    animation: name duration timing-function delay iteration-count direction fill-mode play-state;

/* 示例 */
.container{
    animation:rotate 2s linear 0 infinite ;
}
@keyframes rotate{
  from{
    transform: rotate(0deg);
  }
  to{
    transform: rotate(360deg);
  }
}
/* 或 */
  @keyframes rotate{
    0%{
      transform: rotate(0deg);
    }
    50%{
        transform: rotate(180deg);
    }
    100%{
      transform: rotate(376deg);
    }
  }
