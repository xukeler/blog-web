    # vue-cli4

    • vue-cli是vue的脚手架工具，用来搭建开发环境，把开发所需要的大部分的工具都搭建好。
    • 最新vue-cli的版本到4了，vue-cli是基于webpack构建的。
    • 官方文档

    # 开始搭建

    • 下载全局依赖npm install -g @vue/cli或yarn global add @vue/cli。下载后的版本显示Vue CLI v4.5.11。
    • 使用命令vue create 项目名称创建一个项目，然后会提示是否使用淘宝镜像，国内使用npm下载比较慢，但是使用淘宝镜像会有一些奇怪问题,建议不要用淘宝镜像，大不了科学上网。
    • vue-cli 4会提示搭建vue2.0的模板项目还是vue3.0模板的项目,还是手动配置。

    • 手动配置需要勾选配置项，例如vuex,vue-router等。

    • 选择好配置项，提示使用npm还是yarn，看个人喜好用那个，选择之后就开始下载。

    • 下载完成后，会提示打开项目npm run serve 启动。

    提示

    选择模板构建不会构建vuex，vue-router等工具，需要再另外下载

    • 调用npm run serve就成功启动了，然后就可以正式开发了。
    • 打包还是调用npm run build
