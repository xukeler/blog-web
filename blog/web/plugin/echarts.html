    # echarts

    • 在工作中会遇到一些需求，把数据以图表的形式展示出来，也就是数据可视化，数据展示出来，能够更直观的分析数据规律，有利于公司的运营分析。
    • echarts是百度的一块数据可视化插件，能够将数据展示为各种各样的图表，常见的条形图，线型图，饼图等。
    • 官网

    # 下载使用

    • 使用有两种方式一种是直接通过script标签引入，一种通过npm 下载引入使用。
    <script src="echarts.min.js"></script>
//或
npm install echarts --save

    # 绘制图表

    • 绘制图表需要一个容器设置宽高，通过echarts.init初始化一个echarts实例，再进行一些数据配置。

    • 如果要生成多个图表，需要初始化多个echarts实例，需要不同的容器。

    • echarts里面的一些内容被抽象成组件，例如toolbox工具栏，坐标轴等，可以按需引入，也可以全部引入。

    • echarts实例通过setOption方法进行图表的数据，格式，样式，类型，交互的配置。

    • 一些功能需要参照官方文档实例实现。

    • 一个简单的条形图

    this.myChart = echarts.init(this.$refs.EasyChart);
this.myChart.setOption({
    title: {
        text: '条形图'
    },
    tooltip: {},
    xAxis: {
        data: ['js', 'vue', 'react', 'taro', 'uni-app', 'electron']
    },
    yAxis: {
        min:50,
        max:300,
    },
    series: [{
        name: '熟练程度',
        type: 'bar',
        data: [5, 20, 36, 10, 10, 20],
        itemStyle: {
            // 阴影的大小
            shadowBlur: 200,
            // 阴影水平方向上的偏移
            shadowOffsetX: 0,
            // 阴影垂直方向上的偏移
            shadowOffsetY: 0,
            // 阴影颜色
            shadowColor: 'rgba(0, 0, 0, 1)'
        }
    }]
});

    # 样式设置

    • 对图表进行个性化的样式设,在itemStyle中进行设置，在series中有一个全局的itemStyle，对每个数据单独设置，需要在series里面的data配置itemStyle。
    • 全局可以设置背景色字体颜色，深色或者浅色自由搭配，通过setOption里面backgroundColor设置全局背景颜色，textStyle设置全局字体样式。

    # 管理数据

    • 设置数据的方式有两种，一种是在series里面的data设置，如果设置多个数据在series里面添加多个数据对象。
    • 一种是通过dataSet的source来配置数据映射到图形中，将数据以二维数组的形式来配置，第一行或列配置维度的名称，然后第二行开始是数据。也可以按行的 key-value 形式（对象数组），其中键key表明了 维度名。
    this.myChart = echarts.init(this.$refs.DataSetChart);
this.myChart.setOption({
    backgroundColor:"#fafcfd",
    textStyle:{
        color:"#172f77"
    },
    dataset:{
        source:[
            ["der", '1月', '2月', '3月', '4月'],
            ["男装", 41.1, 30.4, 65.1, 53.3],
            ["女装", 86.5, 92.1, 85.7, 83.1],
            ["童装", 24.1, 67.2, 79.5, 86.4]
        ]
    },
    title: {
        text: 'dataSet',
        show:true,
        textAlign:"left",
        link:"https://www.xukeler.cn",
        target: 'blank' ,
        textStyle:{
            color:"#172f77"
        },
    },
    tooltip: {},
    series: [{
         name: '业绩',
        type: 'pie',
        id: 'pie',
    }]
});

    # 异步加载数据

    • 开发中数据肯定不是写死的，是由后端通过api将数据以json的形式返回，那这个时候我们就需要处理数据，然后在图表中体现出来。
    • 可以在请求完成后拿到数据再调用setOption,或者先画图表，数据为空，请求完成之后再调用setOption，设置data,和series里面的data,这个时候只设置data，不用关注别的配置项。
    //模拟异步加载
setTimeout(()=>{
    this.myChart.setOption({
        xAxis:{
            data:['1月',"2月","3月","4月","5月"]
        },
        series: [{
             name: '业绩',
             data:[90,54,23,89,8]
        }]
    })
},2000)

//初始化
