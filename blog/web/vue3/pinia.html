    # pinia

    • pinia是一个全新的Vue状态管理库，功能相对于vuex更轻量化，体积小，支持Vue2或Vue3但是最好还是在Vue3中使用，Vue2用vuex感觉更好一点。
    • pinia只有state getters actions没有Mutation module，没有模块的概念，只有store,按store分模块。
    • 官网

    # 下载安装

    npm i pinia

    import { createApp } from 'vue'

    # 创建store

    • 项目里store文件没有限制，store之间可以互相引用。
    • 一个store包含state getters actions
    • actions 支持异步async/await。
    • state在页面不能直接解构使用，否则会失去响应性，应该通过storeToRefs(state)来解构。
    • getters和计算属性是一样的。
    import { defineStore } from "pinia";
export const testStore=defineStore("test",{
    state:()=>{
        return {
            count:1,
            age:18
        }
    },
    actions:{
        addCount(){
            // 可以通过this访问
            this.count++
        },
        reduceCount(){
            this.count--
        }
    },
    getters:{
        computedCount(state){
            return `getters count value${state.count}`
        }
    }

})

    # 在页面中使用

    • 首先要引入store，store是一个函数，必须调用使用。
    • 也可以使用storeToRefs解构后使用。
    • 也可以结合computed watch使用。
    <template>
  <div>
    <div class="title">子组件</div>
    <div class="value">
     数值 {{ count }}
    </div>
    <div class="value">
     年龄 {{ age }}
    </div>
    <div class="value">
      数值{{ computedCount}}
    </div>
  </div>
</template>
import { testStore } from "../store/index";
import {storeToRefs} from "pinia"
import { watch } from "vue";
const data = testStore();
const {count,age,computedCount}=storeToRefs(data)
//解构的的数值不能直接操作
watch(count,(val)=>{
    console.log("新的值"+val)
})

    # 修改数据

    • 修改state数据，可以直接修改，也可以通过actions修改，也可以通过store.$patch修改。
    • store.$patch可以接收函数或者对象作为参数,可以修改多条参数。
    import {testStore} from "../../store/index";
const store=testStore()
const add=()=>{
    // store.count+=1;  //直接修改
    store.addCount()  // actions 修改
    // store.$patch({   // store.$patch 对象修改
    //     count:store.count+1,
    //     age:store.age+1
    // })
    // store.$patch((state)=>{    // store.$patch 函数修改
    //     state.count+=1;
    //     state.age+=1
    // })
}
const reduce=()=>{
    // store.count-=1
    store.reduceCount()
    // store.$patch({
    //     count:store.count-1,
    //     age:store.age-1
    // })
    // store.$patch((state)=>{
    //     state.count-=1;
    //     state.age-=1
    // })
}
