    # 录音下载

    • 最近开发的遇到这个需求，前端录制音频并下载，公司旧的项目里面也有封装好的record.js用来前端录音的，但是小bug很多，也不支持下载。 所以查找了一些解决方案，有一个js-audio-recorder插件，效果挺好的满足开发需求。
    • github文档

    # Development

    • 下载
    cnpm install js-audio-recorder --save
    1
    • 使用在组件中引入
    import Recorder from 'js-audio-recorder';
    1
    • 在组件中使用
    • 采样位数和采样率声道都可以动态设置。
    this.recorder1 = new Recorder({
    sampleBits: 16,         // 采样位数，支持 8 或 16，默认是16
    sampleRate: 16000,      // 采样率，支持 11025、16000、22050、24000、44100、48000，根据浏览器默认值，我的chrome是48000
    numChannels: 1,         // 声道，支持 1 或 2， 默认是1
    compiling: false,       // 是否边录边转换，默认是false          
});

    # Demo

    准备录音了

    # Code

    <template>
    <div style="width:100%;height:300px">
        <div>{{text}}</div>
        <div style="margin-top:50px">
          <button @click="startRec">开始录音</button>
          <button @click="stopRec">结束录音</button>
          <button @click="playRec">播放录音</button>
          <button @click="downRec">下载录音</button>
          <!-- <button @click="clearRec">清除示例</button> -->
        </div>
    </div>
</template>

<script>
    import Recorder from 'js-audio-recorder';
    export default {
        data(){
            return{
                recorder:null,
                text:"准备录音了"
            }
        },
        mounted(){
            this.init();
        },
        methods:{
            init(){
                this.recorder = new Recorder({
                    sampleBits: 16,                 
                    sampleRate: 16000,              
                    numChannels: 1,                          
                });                
            },
            clearRec(){
                this.recorder.destroy()
            },
            startRec(){
                this.recorder.start().then(() => {
                    // 开始录音
                    this.text="开始录音"
                }, (error) => {
                    // 出错了
                    this.text="出错了"
                });
            },
            stopRec(){
                this.text="结束录音"
                this.recorder.stop();
            },
            playRec(){
                this.text="播放录音"
                this.recorder.play();
            },
            downRec(){
                this.recorder.downloadWAV("录音文件")
            }
        }
    }
</script>
