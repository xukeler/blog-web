    # 移动端弹窗滚动穿透

    • 最近项目重构在封装移动端弹窗组件(popup)时遇到这个问题，打开弹窗滚动就会出现下面这些问题。
      • 遮罩层滑动，被遮挡的内容部分也会滚动
      • 弹窗的内容部分滚动，被遮挡的内容部分也会滚动
    • 解决方案
      • 用vue开发直接就在标签加@touchmove.prevent阻止默认行为，只要阻止默认行为就能禁止遮罩滚动穿透。
      • 最好是给body加个样式overFlow:hidden，在打开弹窗时hidden，关闭或销毁弹窗组件设置unset。

    WARNING

    需要注意的是，在页面已经发生滚动的前提下，设置overflow后，关闭弹窗再设置unset，页面可能会滚动到顶部。

    <template>
    <div @touchmove.prevent>
    </div>
</template>
