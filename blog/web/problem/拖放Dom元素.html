    # 拖放Dom元素

    • 开发遇到一个小需求，就是定位在页面的一个音乐播放器，在最小化的时候是可以拖放的，拖放顾名思义就是拖动一个元素，移动到指定的位置。
    • 实现的思路也比较简单，监听mousedown，mouseleave，和mouseup时间，鼠标按下的时候打开开关，根据元素所在的偏移值和event对象的clientX，clienY计算元素的定位。鼠标抬起的时候关闭开关，开关的作用就是监听移动只有开关为true的时候才计算。

    提示

    addEventListener监听会对页面性能有影响，离开当前页面需要移除，否则会影响到其它页面。

    # demo

    # code

    <template>
    <div id="drag-container" @mouseleave="DragEnd($event)" @mousemove="DragMove">
        <div id="drag-el" @mousedown="dragStart($event)" @mouseup="DragEnd($event)">

        </div>
    </div>
</template>

<script>
    export default {
        data(){
            return {
                siteX:"",
                siteY:"",
                offsetL:"",
                offsetT:"",
                isDown:false,                
            }
        },
        mounted(){
            window.addEventListener("mouseup",this.DragEnd)
        },
        destroyed(){
            window.removeEventListener("mouseup",this.DragEnd)
        },
        methods:{
        DragMove(event){
            const e = event || window.event 
            let node=document.getElementById("drag-el");
            let parentNode=document.getElementById("drag-container")
            if (this.isDown == false) {
                return;
            }
            let nx = e.clientX;
            let ny = e.clientY;
            let nl = nx - (this.siteX - this.offsetL);
            let nt = ny-this.siteY +this.offsetT;
            let h=parentNode.scrollHeight;
            let w=parentNode.scrollWidth;
            let nodeW=node.scrollWidth
            let nodeH=node.scrollHeight;
            if(nl>=0&&nl<=(w-nodeW)){
                node.style.left = nl + 'px';
            }
            if(nt>=0&&nt<=(h-nodeH)){
                node.style.top =nt + 'px';

            }
        },
        dragStart(e){
            let node=document.getElementById("drag-el")
            this.siteX = e.clientX;
            this.siteY = e.clientY;
            this.offsetL =  parseInt(document.defaultView.getComputedStyle(node,null).getPropertyValue('left'));
            this.offsetT = parseInt(document.defaultView.getComputedStyle(node,null).getPropertyValue('top'));
            this.isDown = true;
            node.style.cursor = 'move';
        },
        DragEnd(){
            this.isDown = false;
            let node=document.getElementById("drag-el")
            node.style.cursor="default"
        },
        }
    }
</script>

<style scoped>
#drag-container{
    width: 100%;
    height: 500px;
    position: relative;
}
#drag-el{
    width: 50px;
    height: 50px;
    border-radius: 50%;
    background-color: skyblue;
    position: absolute;
    top: 5px;
    left: 5px;
}
</style>
