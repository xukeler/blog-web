    # 直播推流和聊天室

    • 最近开发一个简单直播需求,通过前端来配置直播的内容，将后端返回的推流地址打开一个直播页面，进行直播，整个直播的信息都是可配置的， 如下图所示

    • 根据客户的要求，直播选择了腾讯和阿里的两种类型。配置好直播信息之后，后端会根据阿里和腾讯的直播文档规则，返回两个链接，一个直播推流的链接，需要使用直播软件进行推流。一个是观众链接，用户打开观众链接进行观看。
