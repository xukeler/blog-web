    # websocket

    • websocket是HTML5开始提供的一种在单个TCP连接上进行全双工通讯的协议。能够实现客户端和服务器之间互相通信，而以往的http只能客户端向服务器发消息，服务器不能主动向客户端发消息，而websocket就完美解决了这个问题。没有同源限制，客户端可以与任意服务器通信。使用场景也比较多，例如聊天功能，直播功能等。

    # 使用

    • 在web上使用，通过new WebSocket(协议)实例化一个socket对象，只需要建立一次连接，协议标识符是ws如果加密，则为wss。
    • 实例对象包含一些属性和方法。

      • readyState表示连接状态。

        • 0表示还没有连接。
        • 1表示已经建立连接。
        • 2表示连接正在关闭。
        • 3表示连接已经关闭，或者不能打开。

      • url socket的链接。

      • onopen方法，开始建立连接时触发。

      • onmessage客户端收到服务端数据时触发。

      • onerror通信发生错误时触发。

      • onclose连接关闭时触发。

      • send(data)向服务端发送消息。

      • close()关闭连接。

    • 使用nodejs-websocket创建一个简单的socket协议服务，在本地启动模拟客户端和服务器的通行。

    # 客户端

    <template>
  <div>
    <button @click="send(1)">天气</button>
    <button @click="send(2)">日期</button>
    <div >{{content}}</div>
  </div>
</template>

<script>
  export default {
    data(){
        return{
            content:"",
            socket:null,
        }
    },
    mounted(){
        this.socket =new WebSocket("ws://192.168.1.114:3001");
        this.socket.onmessage=(evt)=>{
         this.content=evt.data
        }
    },
    methods:{
        send(code=0){
            if(this.socket){
                this.socket.send(code)
            } 
        }
    }
  }
</script>

    # 服务端

    var socket =require("nodejs-websocket");
var port=3001;
var server=socket.createServer((con)=>{
    con.on('error',(code)=>{
        console.log("错误",code)
    })
    con.on('text', (result)=> {
        var string="";
        if(result==1){
            string="天气不错"
        }else{
            string=new Date().toLocaleDateString()
        }
        con.send(string)
        console.log('接收消息', result)
      })
    con.on('close',(code)=>{
        console.log('关闭',code)
    })
    con.on('connect', (code)=> {
        console.log('开启连接', code)
      })
})
server.listen(port,()=>{
    console.log("服务启动监听端口"+port)
})
