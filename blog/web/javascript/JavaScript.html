    # 1.JavaScript是一种什么类型的语言？

    • JavaScript是一种弱类型的脚本语言，主要用于web编程的语言，是一个不完全面向对象的语言。

    # 2.JavaScript主要组成

    • JavaScript是由三部分组成DOM，BOM，ECMAScript（核心）

    # 3.什么是DOM

    • DOM是文档对象模型（Document Object Model,，）的缩写，当网页被加载时，浏览器会创建页面的文档对象模型，DOM可以理解为一系列功能的集合，是针对 XML 但经过扩展用于 HTML 的应用程序编程接口，也就是通俗讲的api，通过这些功能实现对html文档的动态操作，DOM 是为了操作文档出现的 API，document 是其的一个对象。

    注意

    DOM并不是javascript所特有的，别的语言也有DOM，只是在浏览器中，DOM是javascript重要组成部分。

    # 4.什么是BOM

    • BOM是Browser Object Model）的缩写，是指浏览器对象模型。它的功能就是实现JavaScript和浏览器的交互，BOM 是为了操作浏览器出现的 API，window 是其的一个对象。

    # 5.什么是ECMAScript

    • ECMAScript是一个标准，这个标准针对JavaScript语言，ECMAScript描述了以下内容：语法、类型、语句、关键字、保留字、运算符、对象等。

    # 6.JavaScript有几种数据类型？

    • 简单数据类型：null，boolean，number，string，undefined。
    • 复杂数据类型：object，

    提示

    object类型包含array(数组)，object（对象） 对象就是属性和方法的集合叫对象。

    • 1.null 代表的空的意思。

    • 2.boolean布尔类型只有两个值true或false，一般用于开关判断。

    • 3.number就是数值类型，包括各种进制的数值。

    • 4.string字符串类型，被单引号或双引号包裹的字符。

    • 5.undefined类型代表变量没有值或者找不到变量。

    • 6.object对象类型(引用类型)有自己的属性和方法，例如函数(function)数组(array)

    提示

    引用类型:引用类型是指有多个值构成的对象,也就是对象类型，简单理解就是复制一个对象的，改变其中的一个属性，原对象的属性也会改变。

    # 8.字符串对象属性和常用方法

    • length属性返回字符串的长度。
    • charAt()可以返回指定索引的字符。
    • concat()链接两个字符串，返回连接后的字符串。
    • indexOf()获取字符第一次的索引。
    • lastIndexOf()获取字符最后一次出现的索引。
    • replace()删除替换字符串,不会改变原字符串，返回改变后的字符串。
    • split()以指定的字符串切割成数组。
    • slice()截取制定索引之间的字符串并返回,不改变原始字符串，-1为倒数第一个字符以此类推。
    • substr()从指定索引截取指定长度的字符串,第一个参数是索引，第二个是长度。
    • substring()截取两个索引之间的字符串,包含起始索引，不包含结束索引。
    • toUpperCase(),toLowerCase()转换大小写。
    • trim()移除字符串首尾的空格，中间的不能移除。

    # 9.数组对象属性和常用方法

    • length属性返回数组的长度。
    • concat(),连接两个或更多的数组，并返回新的数组,不改变原始数组。
    • join()，将数组以指定分隔符连接成一个字符串,不改变原始数组。
    • pop()删除并返回数组的最后一个元素,改变原始数组。
    • push()从数组的最后一个开添加,返回新数组长度，改变原始数组。
    • shift()从数组头部开始删除,返回删除的元素,改变原始数组。
    • unshift()从数组的头部开始添加。并返回新的长度,改变原始数组。。
    • splice()删除替换添加数组元素。
    • sort()数组元素排序。
    //创建数组
let arr=[]
or
let arr=new Array();
//length
var arr=[1,2,3]
console.log(arr.length)//3

//concat()

let arr2=[4,5,6]
console.log(arr.concat(arr2),arr,arr2)// [1, 2, 3, 4, 5, 6]   [1, 2, 3]    [4, 5, 6]

//join()
let arr=[1,2,3]
console.log(arr.join("+"),arr)//1+2+3   [1, 2, 3]

//pop()

let arr=[1,2,3]

//shift()

console.log(arr.shift(),arr)//1  [ 2,3]

//unshift()
let arr=[1,2,3]
console.log(arr.unshift(4),arr)//4   [4,1, 2,3]

//splice()
let arr=[1,2,3]
console.log(arr.splice(1,1),arr)//delete [2]  [1, 3]
console.log(arr.splice(0,2,4),arr)//change  [1, 2]  [4, 3]
console.log(arr.splice(0,2,4),arr)//add[1, 9, 2, 3]

//sort()排序
var arr=[3,2,6,2,5]
arr.sort((a,b)=>{
  if(a>b){
    return 1
  }else{
    return -1
  }
})

    # 10.Object常用属性和方法

    • Object.create()使用指定的原型对象和属性创建一个新对象。
    • Object.assign(target,source1,source2..)将所有可枚举属性的值从一个或多个源对象复制到目标对象。它将返回目标对象。
    • Object.defineProperty()给对象添加一个属性并指定该属性的配置。
    • Object.keys()返回一个所有元素为字符串的数组，其元素来自于从给定的object上面可直接枚举的属性。
    • Object.values()返回一个给定对象自身的所有可枚举属性值的数组。

    # 11.Math常用方法

    • Math.ceil()对一个数进行向上取整,Math.ceil(x) 返回大于等于 x并且与x最接近的整数
    • Math.floor()对一个数进行向下取整,Math.floor(x)返回小于等于 x 并且与 x最接近的整数
    • Math.round()对一个数进行四舍五入取整,Math.round(x)返回最接近 x的整数
    • Math.random()返回介于 0 ~ 1 之间的一个随机数，不包括 0 和 1。
    • Math.max()返回指定的数中带有较大的值的那个数,Math.max(x1,x2,.....)返回x1,x2,.....中带有最高值的数字，如果有某个参数为 NaN，或是不能转换成数字的非数字值，则返回NaN。
    • Math.min()返回指定的数中带有较小的值的那个数,Math.min(x1,x2,......)返回 x1,x2,.....中带有最小值的数字，如果有某个参数为 NaN，或是不能转换成数字的非数字值，则返回 NaN。
    Math.ceil(3.2);//4
Math.floor(3.7);//3
Math.round(3.5);//4
Math.trunc(123.123);//123
Math.random();//一个随机数
Math.max(3,6,6.2);//6.2
Math.min(3,4,34);//3

    # 12.Number常用方法

    • isFinite(i)判断是否是有限大的数。
    • isNaN(i)判断是否是 NaN。
    • isInteger(i)判断是否是整数。
    • parseInt(str)将字符串转换为对应的数值。
    • toString()把数字转换为字符串，使用指定的基数。

    # 13.Window对象常用属性和方法

    • innerHeight返回窗口的文档显示区的高度。
    • innerWidth 返回窗口的文档显示区的宽度。
    • Location.href返回完整的URL。
    • reload()重新载入当前文档。

    # 14Dom对象常用属性和方法

    • addEventListener()事件监听。
    • appendChild()为元素添加一个新的子元素，从后面插入。
    • className设置或返回元素的class属性。
    • clientHeight在页面上返回内容的可视高度（不包括边框，边距或滚动条）
    • clientWidth在页面上返回内容的可视宽度（不包括边框，边距或滚动条）
    • cloneNode()克隆节点。
    • focus() 设置文档或元素获取焦点
    • getAttribute()返回指定元素的属性值
    • hasAttribute() 如果元素中存在指定的属性返回 true，否则返回false。
    • innerHTML设置或者返回元素的内容。
    • insertBefore() 现有的子元素之前插入一个新的子元素
    • offsetHeight返回任何一个元素的高度包括边框和填充，但不是边距
    • offsetWidth返回元素的宽度，包括边框和填充，但不是边距
    • offsetLeft返回当前元素的相对水平偏移位置的偏移容器
    • offsetTop返回当前元素的相对垂直偏移位置的偏移容器
    • removeAttribute()从元素中删除指定的属性
    • removeChild() 删除一个子元素
    • scrollHeight返回整个元素的高度（包括带滚动条的隐蔽的地方）
    • scrollLeft返回当前视图中的实际元素的左边缘和左边缘之间的距离
    • scrollTop返回当前视图中的实际元素的顶部边缘和顶部边缘之间的距离
    • scrollWidth返回元素的整个宽度（包括带滚动条的隐蔽的地方）
